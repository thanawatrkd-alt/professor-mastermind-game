# ทรชนคนปล้นไทย: Mastermind v33

เกมจิตวิทยาแบบโต้ตอบที่คุณสามารถเล่นกับอาจารย์ (Professor) ได้

## ฟีเจอร์

- 🎭 ตัวละครอาจารย์ที่ฉลาดและลึกลับ
- 🧠 ตอบคำถามด้วยจิตวิทยา
- 📊 Accuracy ที่เปลี่ยนแปลงตามเนื้อเรื่อง
- 🎨 ดีไซน์สวยงาม

## วิธีใช้

1. เปิดไฟล์ `index.html` ด้วย Browser
2. พิมพ์คำถามในช่องพิมพ์
3. คลิก "ส่ง" เพื่อส่งคำถามไปยัง AI
4. รับคำตอบจากอาจารย์

## API Key

ต้องใส่ OpenAI API Key ในไฟล์ `index.html` บรรทัด 163:
```javascript
const API_KEY = 'your-openai-api-key-here';
```

## เนื้อเรื่อง

เกมนี้อิงจากเรื่อง "ทรชนคนปล้นไทย" ซึ่งเป็นเกมปล้นธนาคารที่ใช้จิตวิทยาในการวางแผน
